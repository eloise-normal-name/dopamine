## Layer Structure for Final Concept

When creating the layered final:
1. **Sky** - Deep blue-purple night sky
2. **Far background** - Distant city/structures
3. **Background market** - Distant stalls and depth
4. **Midground** - Main market area (machine placement zone)
5. **Foreground** - Framing elements (bokeh, near details)
6. **Lighting overlay** - Glow, neon light effects
7. **Atmosphere** - Fog, steam, ambient particles

## Implementation Assets

From approved concepts, extract:
- Background layer sprites (static or parallax layers)
- Neon sign SVGs or sprites
- Lighting overlay effects
- Atmospheric particle systems
- Color grading LUT or palette
- Ambient animation references

## Technical Considerations

- **Parallax scrolling**: Prepare layers at different scroll speeds
- **Performance**: High-detail vs simplified versions
- **Responsive**: Works at different aspect ratios (16:9, 9:16, 4:3)
- **Accessibility**: Reduced motion version planning
