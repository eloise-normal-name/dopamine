## Technical Requirements

- **Format**: PNG for individual components, JPG for complex assemblies
- **Resolution**: 1200x1200 for single components, 2400x3000 for sheets
- **Style**: Technical illustration with artistic polish
- **Detail level**: High - these are production reference images
- **Annotations**: Include labels, dimensions, material callouts
- **Color mode**: RGB, sRGB
- **File size**: Individual < 400KB, sheets < 1MB

## Drawing Specifications

Each component study should show:
- **Clear construction** - How parts fit together
- **Material properties** - Surface finish, reflections
- **Multiple states** - Idle, active, winning (where applicable)
- **Scale reference** - Relative sizes and proportions
- **Edge treatments** - Bevels, curves, seams
